## 🛠️ Architecture & Core Features

This platform is custom-built to maximize user experience and frontend efficiency, departing from standard stock templates:

*   **Persistent Favorites System:** Built-in tracking utilizing local storage, enabling users to instantly curate a personalized dashboard.
*   **Dynamic View Transitions:** Fluid navigation transitions utilizing asynchronous DOM updates for a seamless single-page application (SPA) feel.
*   **Unified Changelog Tracking:** Integrated version logging to transparently document asset changes, game additions, and layout revisions.
*   **Web-Optimized Hosting:** Tailored for GitHub Pages execution, utilizing structural code fixes to fetch split WebGL engine components efficiently.
